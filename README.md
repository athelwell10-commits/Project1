# LeakGuard

## Description
LeakGuard is a water leak detection app designed to provide real-time alerts and monitoring. Utilizing advanced sensor technology, the app helps prevent water damage and conserve water resources.

## Installation Instructions
To install the app, first clone the repository:

```bash
git clone https://github.com/athelwell10-commits/Project1.git
cd Project1
```

Then, run the following commands:

```bash
npm install
```

## How to Run the App
After installation, you can start the app using:

```bash
expo start
```

This command will launch the Expo development server.

## Project Structure
- `src/` : Contains all source files for the application.
- `components/` : Includes reusable components used throughout the app.
- `screens/` : Contains all screen components for the navigation.
- `assets/` : Includes images and other static resources.

## Features
- Real-time leak detection alerts
- User-friendly interface
- Detailed leak history
- Customizable settings
- Integration with home automation systems

## Troubleshooting
1. If you encounter issues with dependencies, ensure you have the latest version of Node.js and npm.
2. For problems starting the app, check your internet connection and ensure Expo CLI is properly installed.

If you continue to face issues, consider searching in forums or opening an issue on GitHub.
